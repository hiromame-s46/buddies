Buddies profile は、櫻坂46を応援する人のためのプロフィールサービスです。 自分の「My推し」や「NEXT LIVE」を記録し、同じ公演へ行く人や共通点のある人を見つけられます。 このページはガイドとして単独で読める記事です。

01 / ABOUT

まず知っておきたい、できること

Buddies profile は単なる自己紹介ページではありません。推しの情報とライブの予定を起点に、 現地での交流や、あとから振り返る思い出までを一つのプロフィールにまとめられます。

My推しを伝える

推しメン、推し始めた時期、好きな曲、興味のあることをプロフィールに掲載。

ライブで見つける

参加予定の公演と座席を登録し、同じライブに参加するBuddiesを確認。

プロフィールを交換

検索・公開プロフィール・QR交換を使って、その場でもオンラインでもつながれる。

思い出を残す

My Buddies history に参加ライブや思い出を積み重ねて振り返れる。

最初は全部埋めなくても大丈夫 表示名と推し、次のライブ予定だけでもプロフィールは始められます。使いながら少しずつ育てていくのがおすすめです。
02 / GET STARTED

アカウントを作ってログインする

はじめにアカウントへログインします。すでに SakuLabo のアカウントを利用している場合は、 同じアカウントで Buddies profile にログインできます。

  1. ログイン画面を開く

    トップ画面のログイン導線から、ログインまたは新規登録を選びます。

  2. 新規の場合はアカウントを登録

    案内に沿って必要事項を入力します。すでにアカウントがあればログインへ進みます。

  3. MyPage を開く

    ログイン後は下部メニューの「MyPage」から、自分のプロフィール編集を始められます。

アカウント情報について アカウント設定でユーザー名やパスワードを変更した場合、SakuLabo 側のログイン情報にも反映されます。
03 / NAVIGATION

4つのメニューを使い分ける

ログイン後の主な操作は、画面下部のメニューを基点にすると迷いません。

Home おすすめのBuddiesや自分のプロフィールの概要、同じ NEXT LIVE に参加する人を確認します。
Search 表示名、推しメン、タグ、好きな曲、ライブなどを手がかりにBuddiesを探します。
Next Live ライブ一覧と各公演のページへ移動し、参加予定・座席・参加者を管理します。
MyPage プロフィールの編集、履歴、連携、QR交換、共有など自分に関わる操作を行います。
04 / PROFILE

自分らしいプロフィールを作る

MyPage のプロフィール編集には、自己紹介を整えるための項目がまとまっています。 共通点がわかるほど、検索やライブ会場で会話を始めやすくなります。

基本情報

まずは表示名を決め、必要に応じてプロフィール画像、自己紹介、居住エリアや年代などを設定します。 画像を設定すると、検索結果やプロフィール上であなたを見つけてもらいやすくなります。

My推し

推しにまつわる情報を入力すると、プロフィールの中心となる「My推し」欄に反映されます。

推しメン 推し始めた時期 好きな曲 興味・タグ

推しメンは複数設定でき、中心となる推しを伝えられます。好きな曲や興味のタグは、 似た楽しみ方をするBuddiesと出会う手がかりになります。

SNSリンク

X、Instagram、Threads、TikTok、YouTube など、自分が公開したいSNSをプロフィールに載せられます。 交流したい範囲に合わせて、公開して問題のないアカウントだけを登録してください。

プロフィール画像を未設定の場合 ライブ参加者一覧では、推しメンを設定していると対応するメンバー画像がアイコンとして表示されます。 自分で選んだプロフィール画像を登録することもできます。
05 / NEXT LIVE

参加するライブと座席を登録する

「NEXT LIVE」は、これから参加する公演をプロフィールに表示し、同じ会場へ向かうBuddiesを探すための機能です。 ライブ前に設定しておくと、当日の交流にも役立ちます。

Next Live→ 公演を選択→ 参加予定を登録→ 座席を入力

ライブ一覧で見るもの

  • ライブ名、日付、会場と、自分が登録した座席情報を確認できます。
  • 参加予定にした公演には参加状態が表示され、参加人数も把握できます。
  • 目的の公演を選ぶと、各ライブの詳細ページへ進みます。

各ライブページでできること

公演ページのヒーロー部分にはライブ名、会場、自分の座席、参加予定人数が表示されます。 日付の左右にある矢印を使うと、前後の日程の公演へ移動できます。 右上のリンクアイコンからは、そのライブの公式案内ページを開けます。

座席を登録・編集する

  1. 「参加予定・座席を編集」を選ぶ

    参加予定の登録と座席入力を行う編集画面が開きます。

  2. 座席の区分を入力する

    階、アリーナ／スタンドなどのエリア、ブロック、必要に応じて座席番号を入力します。

  3. 保存して表示を確認する

    登録した座席は、ライブ一覧と各公演ページで会場情報の下に表示されます。

近い席のBuddiesを見つける

公演ページでは同じライブに参加するBuddiesを一覧できます。 「自分の席と近いBuddiesを優先して表示」を有効にすると、登録した座席情報をもとに 近い可能性がある人が先に並びます。該当しない人が消えるのではなく、全員を見たまま順序だけが変わります。

座席情報を公開する前に 交流のきっかけとして便利な一方、公開範囲には注意が必要です。表示したい範囲で入力し、 ライブ終了後は必要に応じてプロフィール内容を見直しましょう。
06 / CONNECT

Buddiesを探す・交換する

Search から探す

Search では、表示名や推しメン、好きな曲、タグ、参加ライブなどをキーワードにしてプロフィールを探せます。 フィルターを組み合わせれば、「同じ推し」や「同じ公演に参加」のような共通点から絞り込めます。

  • 気になるプロフィールを開き、My推しやNEXT LIVEを確認します。
  • 公開されているSNSがあれば、プロフィールから交流先を確認できます。
  • お気に入り登録を使って、あとで見返したいBuddiesを残せます。

会場ではQRで交換

MyPage のQR交換機能は、ライブ会場でプロフィールを見せ合うときに便利です。 「見せる」で自分のQRを表示し、「読み取る」で相手のQRを読み取ります。

見せる

相手に読み取ってもらうQRを表示します。安全のためQRは一定時間で更新されます。

読み取る

アプリ内の読み取りを使うと、交換したBuddiesとして記録できます。

カメラアプリで読む場合 端末標準のカメラでQRを開くと公開プロフィールの閲覧が中心になります。 交換として履歴に残したいときは、Buddies profile 内の読み取り機能を使います。
07 / SHARE

公開プロフィールを共有する

あなたの公開プロフィールには、入力した内容に応じて NEXT LIVE、My推し、Bio、My Buddies history、 各種連携の記録やSNSリンクなどが表示されます。公開リンクを送れば、ログインしていない相手にも 自分のプロフィールを紹介できます。

共有前のチェック

  • 表示名、プロフィール画像、自己紹介が意図した内容になっているか確認します。
  • NEXT LIVE や座席情報を公開して問題ないかを確認します。
  • SNSリンクは交流に使いたいアカウントだけを掲載します。

プロフィール画面の共有機能からリンクを共有したり、X向けの投稿を作成したりできます。 投稿する前に、文章と公開されるプロフィール内容を一度確認すると安心です。

08 / MEMORY & LINK

履歴を残し、関連サービスとつなぐ

My Buddies history

My Buddies history には、Buddiesとの思い出や過去に参加したライブを残せます。 ライブが増えるほど、自分の応援の歩みを振り返れるページになっていきます。 履歴画面ではライブや画像の有無などを手がかりに探せます。

各種連携

プロフィール編集の「各種連携」から、利用しているサービスの記録をプロフィールに表示できます。

SakuLabo お気に入りにしたさくみみやブログを、選択してプロフィールに表示できます。
SakuMap 獲得したスタンプを表示し、訪れた場所の記録をプロフィールに加えられます。
SakuMV Quiz カテゴリごとの自己ベストやランキングなど、クイズの記録を表示できます。

Today's 櫻坂

日々の予定やブログ、さくみみ、楽曲・MV・過去の出来事などを確認できるページです。 SakuLabo／Buddies の連携ログインを利用すると、対応するコンテンツのお気に入り操作にもつながります。

09 / SETTINGS

設定と安心して使うためのポイント

アカウント設定ではログインに関わる情報の変更やログアウトができます。 不特定多数へ共有するプロフィールだからこそ、入力する情報は自分で管理できる範囲にとどめましょう。

公開内容を見直す

ライブ予定、座席、SNSなどは、公開リンクを共有する前後に確認。

会場での交流

QR交換や座席を使った交流は、無理のない範囲で安全に楽しむ。

ログイン情報

パスワードは他サービスと使い回さず、第三者には伝えない。

困ったとき

操作に迷った場合は、サイト内の Help や案内ページを確認。

最初の5分でやること

  1. 表示名とプロフィール画像を設定する

    プロフィールの印象を整え、見つけてもらいやすくします。

  2. My推しと好きな曲を入力する

    同じ推し・同じ好みのBuddiesとの接点になります。

  3. 参加予定のNEXT LIVEを登録する

    予定があれば公演を選び、必要に応じて座席も追加します。

  4. 公開プロフィールを確認する

    相手から見える内容を確認してから、検索や共有を楽しみます。

10 / FAQ

よくある疑問

プロフィールは全項目を入力しないと使えませんか？

いいえ。まずは表示名や推しだけでも始められます。ライブ予定やSNS、履歴は必要になったタイミングで追加できます。

座席を登録すると、近い席の人だけが表示されますか？

近い席のBuddiesを優先表示する設定は、一覧の並び順を調整する機能です。他の参加者が一覧から消えるわけではありません。

ライブの公式ページはどこから開けますか？

各ライブ詳細ページのヒーロー右上にある薄いリンクアイコンから、用意されている公式案内ページを開けます。

QRを見せるだけで交換履歴に入りますか？

交換として記録したい場合は、Buddies profile 内のQR読み取り機能を使って読み取ります。標準カメラで開く場合はプロフィール閲覧が中心です。

SakuLaboを使っている場合、新しいアカウントが必要ですか？

すでに SakuLabo のアカウントを利用している場合は、そのアカウントでログインできます。
